# sql-iniciantes-notebooklm
Este repositório faz parte de um desafio da DIO e mostra minha jornada aprendendo SQL do zero com apoio do NotebookLM. Aqui organizei conteúdos, pratiquei com IA e reuni resumos e exercícios. A ideia é servir como um guia simples para iniciantes e mostrar minha evolução durante o aprendizado.
